Khilgaon Govt High School Admission Application

1.At First, go to this site gsa.teletalk.com.bd and select your division.
2.go to next step and select your District/Zilla, Thana/Upazilla, School.
3.Fill up the application form with required Info.
4.Don’t forget to fill up your quota Information (if any).
5.Upload a color photograph within 300 × 300 pixel in JPG format (Maximum100 KB).
6.Finally submit your online application form.
7.Download your applicant copy & print it out in color.
8.then pay the admission test fees by using “User ID and PIN”. For this we have to write down the user ID and PIN number or save it somewhere. This user ID and PIN number will be required during the next activity.
Be sure to check all the information before filling and submitting the form online. Because all the information you fill in will be used in all subsequent activities of the student. Therefore, care must be taken while filling the form.

                                                               How to Pay Fees

To pay admission test fees by Teletalk, follow the instruction below.

SMS #1: GSA User ID send to 16222

For Example: GSA ABCDEF send to 16222

Reply Message: Will be get students name with pin number

SMS #2: GSA Yes PIN send to 16222

For Example: GSA Yes 123456 send to 16222

                                How to Recovery School Admission User ID and PIN?

We often lose or forget important information such as passwords. If you ever forget the user ID or PIN number then the way to get back the user ID and PIN number is,

  • If know User ID: GSA <Space> Help <Space> User <Space> User ID & send to 16222
  • If know PIN Number: GSA <Space> Help <Space> PIN <Space> PIN No & send to 16222
